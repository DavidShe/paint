# Paint
This project is a paint-like webapplication written in Javascript, HTML5.
I’m using Bootstrap for some styles.

## Features
* Pen
* Text
	* Multi-fonts
* Line
* Circle
* Rectangle
* Paint bucket
* Rubber
* Copy/Paste

## Extra features
* Layers (click to roll back)
* Undo (CMD+Z)
* Import (drag’n’drop)
* Export (CMD+S)
* Zoom (scroll)

Every tools have thickness and color settings.
